HTML and CSS

HTML

HTML stands for HyperText Markup Language.
HTML is written in HTML elements consisting of tags
enclosed in angle brackets. HTML elements form the
building blocks of a web page. A web browser can read
HTML files and compose it into web pages.

CSS

CSS stands for Cascading Style Sheets. Web browsers can
also refer to CSS to define the look and layout
of text and other materials. HTML and CSS standards,
encourages the use of CSS over explicit presentational
HTML.

Bootstrap

Bootstrap has tools for creating websites and
web applications. It contains HTML and CSS-based
design templates buttons, navigation and other
interface components, as well as optional JavaScript
extensions.

MelonJS

melonJS integrates the popular Tiled map format,
allowing to easily design levels.melonJS is an open-source
HTML5 based community game engine.

jQuery

jQuery is a cross-platform JavaScript library designed
to simplify the client-side scripting of HTML.jQuery's
syntax is designed to make it easier to navigate a
document, select DOM elements, create animations,
handle events, and develop Ajax applications.
